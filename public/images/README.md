# Ofer's Atelier Images

Use this folder for production photography.

- `chef-ofer-real.jpg` is the real chef portrait selected from the original Wix source and should remain the main brand image.
- `gallery/` contains the current Wix product and experience photos used by the website.
- Add future Facebook, Instagram, and professional shoot images to `gallery/`, then update `src/lib/brand.ts`.
- Keep filenames lowercase with hyphens, for example `private-fire-feast.jpg`.
- Recommended sizes: 1600px wide for hero/about images, 1200px wide for gallery and experience cards.
